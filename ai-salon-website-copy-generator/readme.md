# AI Salon Website Copy Generator

## Project Overview

This project demonstrates how prompt engineering and AI tools can generate conversion-focused website copy for local salon businesses.

The goal was to design a reusable prompt system capable of generating:
- homepage copy
- service page content
- CTA sections
- tone adaptations
- SEO-focused content

The project simulates how modern web agencies can use AI-assisted workflows to help local businesses improve website communication and customer conversions.

Road No. 12, Banjara Hills, Hyderabad, Telangana

Business Type:
Salon and Beauty Studio

Target Audience:

---

# Problem Statement

Many local salons struggle with:
- weak website copy
- unclear service descriptions
- generic content
- poor call-to-actions
- lack of customer-focused messaging

Hiring professional copywriters can be expensive for small businesses.

This project demonstrates how AI-powered prompt engineering can help generate website-ready content efficiently.

---

# Solution

A structured AI prompt framework was designed to generate:
- homepage website copy
- services page descriptions
- trust-building CTAs
- tone-adapted content
- SEO-focused website text

The prompts were designed to be reusable for multiple local businesses.

---

# Prompt Framework

## Homepage Prompt
Generates:
- hero headline
- subheadline
- intro section
- trust-building content

## Services Prompt
Generates:
- service descriptions
- benefits
- customer-focused explanations

## CTA Prompt
Generates:
- booking CTAs
- urgency CTAs
- consultation prompts
- trust-building website sections

## Tone Adaptation Prompt
Rewrites content in:
- friendly tone
- professional tone
- confident modern tone
